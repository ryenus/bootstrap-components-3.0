Editable

Editable enables user click element to edit it.

Example

This is editable title

Bootstrap Component 3.0 is INSTEASTINT(editable) !

This editable pre content.
<h1 data-editable>This is editable title</h1>

Options

Options should be passed via data attributes. Append the option name to data-, as in data-editable="".

Name Type Default Description
editable string null Set target element be editable. Set value as html will enable editable input box support html format content.

Events

Event Type Description
change Trigger when user modify content. 
$('#editableElement').change(function () {
    // do something...
})
