Checkbox

Example


<span data-toggle="checkbox" class="checkbox btn btn-default"></span>
<span data-toggle="checkbox" class="checkbox btn btn-default" disabled></span>

Options

Options should be passed via data attributes. Append the option name to data-, as in data-to="".

Name Type Default Description
to element null Set target element value when checkbox clicked. (Suggest setting to <input> element)
checkbox-all string null Set this select box as Select All functional. Set by the region name will control all the checkbox which are in the same region.
checkbox-entity string null Support this select box answer the Select All call. Set the region which you want to be as the same group region. 


<span data-toggle="checkbox" class="checkbox btn btn-default" data-checkbox-all="rg1"></span> Select All
<span data-toggle="checkbox" class="checkbox btn btn-default" data-checkbox-entity="rg1"></span> Checkbox A
<span data-toggle="checkbox" class="checkbox btn btn-default" data-checkbox-entity="rg1"></span> Checkbox B

"Select All" is also enabled on original input element

You can also set data-checkbox-all and data-checkbox-entity to the <input type="checkbox" /> element. "Select All" will also works.

Methods

.checkbox(option)

Set checkbox value.

$('#myCheckbox').checkbox({
    checked: true
})

Events

Event Type Description
change Trigger when checkbox clicked. It will also trigger change event for data-to element. 
$('#myCheckbox').change(function () {
    // do something...
})
